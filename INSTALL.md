# krsHubSwap - Готовый JAR файл

## 📥 Как установить:

1. **Скачай JAR файл:**
   - Перейди в папку `releases` в этом репо
   - Скачай `krshubswap-1.0.0.jar`

2. **Помести в Labymod:**
   ```
   .minecraft/labymod/addons/krshubswap-1.0.0.jar
   ```

3. **Перезагрузи игру** - готово! 🎮

## 🎯 Команды:

```
/lt <номер>     - телепортироваться на лайт анархию
/an <номер>     - альтернативный алиас
```

## ⚙️ Алиасы и конфигурация:

Аддон создает файлы конфига в папке `.minecraft`:
- `krshubswap-aliases.txt` - алиасы команд
- `krshubswap-ranges.txt` - диапазоны анархий

## 📋 Диапазоны (по умолчанию):

- **Соло:** 1-15
- **Дуо:** 16-31
- **Трио:** 32-47
- **Кланы:** 48-66

Можно менять в конфиге!
